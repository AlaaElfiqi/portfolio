Projects

Selected robotics, AI, and research projects.

Driver Attention System using ChatGPT, VR, and EEG

Designed VR-based driving experiments using CARLA and DReyeVR with EEG, GSR, heart rate, and conversational AI in Arabic to study stress and driving performance. Applied fuzzy logic, questionnaire analysis, and multimodal sensing for evaluation.

Rescue Robot

Designed a tracked rescue robot for rough terrain and disaster response. Built the chassis, damping system, and robotic arm in SolidWorks and validated the design through stress analysis.

View Report

IRC Competition Robots

Developed both manual and autonomous competition robots. Designed the full mechanical system, gripper mechanism, and H-bridge control circuits to support task execution and object handling.

EEG Emotion Recognition using Machine Learning

Built an end-to-end AI pipeline for EEG preprocessing, feature extraction, and classification using models such as SVM and Random Forest for emotion recognition.

UAV Design and Control

Designed a UAV in SolidWorks with a focus on structural optimization and aerodynamics, and implemented flight control using ArduPilot for stable navigation.

Colour Sorting by Robotic Arm

Built a color sorting system using Python, Arduino, image processing, and robotic arm kinematics to detect and sort objects automatically.
